Q3. Scenario-Based
You received a dataset with 30,000 records, but you found inconsistencies in column formats (e.g., date formats, string capitalization) and some key fields are missing. Describe how you would clean and prepare the data for analysis.
First, I would start by scanning the dataset to get a general understanding of the structure and types of inconsistencies. For the column format issues, such as inconsistent date formats or string capitalization, I would apply standard formatting functions to unify them — for example, converting all dates to ISO format and applying lowercase or title case to text values.
Next, I’d address missing values by checking how critical the missing fields are. If the missing values are in non-essential columns or represent a very small percentage of the data, I might drop those rows. Otherwise, I’d use appropriate imputation techniques such as filling with the mean, median, or most frequent value, depending on the context.
After cleaning, I would validate the integrity of the dataset using basic data profiling tools or functions to ensure there are no remaining inconsistencies. This includes checking for duplicates, verifying data types, and reviewing summary statistics. Once the data is clean and consistent, I’d save the cleaned version and proceed with the analysis.
